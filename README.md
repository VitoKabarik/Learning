## <a id="masks">Описание модуля masks</a>

Модуль содержит две функции: одна зашифровывает номер банковской карты клиента, а другая - номер его лицевого счёта.

## <a id="processing">Описание модуля processing</a>

Модуль содержит две функции: одна ищет в предложенном списке транзакции с определённым состоянием, а другая сортирует транзакции по дате.

## <a id="widjet">Описание модуля widjet</a>

Модуль содержит две функции: одна определяет, имеет она дело с номером банковской карты или же номером лицевого счёта,
и зашифровывает его соответствующим образом, используя функции из модуля masks. Вторая функция принимает строку с датой и делает дату нагляднее.

## <a id="generators">Описание модуля generators</a>

Модуль содержит три функции-генератора: первая по запросу выдаёт транзакции из предложенного списка, совершённые в указанной валюте, вторая по запросу показывает тип транзакций, а третья генерирует номера карт в указанном интервале.

## <a id="decorators">Описание модуля decorators</a>

Модуль содержит декоратор log, который отслеживает состояние декорируемой функции. При нормальном завершении её работы декоратор сообщает время начала действия функции, её имя, а так же информирует о том, что она успешно завершила свою работу. Если же выполнение функции привело к ошибке, декоратор сообщает время начала действия, имя, тип возникшей ошибки, пояснение и входные параметры.
Отчёт записывается в файл, если декоратор получает его имя, или в консоль, если имени нет.

## <a id="utils">Описание модуля utils</a>

Модуль содержит функцию, преобразующую json-файл по указанному пути в строку Python.

## <a id="external_api">Описание модуля external_api</a>

Модуль содержит функцию, которая получает на вход транзакцию и возвращает сумму этой транзакции в рублях.
