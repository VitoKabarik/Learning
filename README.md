## <a id="masks">Описание модуля masks</a>

Модуль, содержащий две функции: одна зашифровывает номер банковской карты клиента, а другая - номер его лицевого счёта

## <a id="processing">Описание модуля processing</a>

Модуль, содержащий две функции: одна ищет в предложенном списке данные с определённым статусом, а другая сортирует данные по дате

## <a id="widjet">Описание модуля widjet</a>

Модуль, содержащий две функции: одна определяет, имеет она дело с номером банковской карты или же номером лицевого счёта,
и зашифровывает его соответствующим образом, используя функции из модуля masks. Вторая функция принимает строку с датой и делает дату нагляднее.
